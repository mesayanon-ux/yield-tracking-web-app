# Yield Web App V4.8.8 - All Operation Detail

Based on V4.8.7.

Updates:
- Operation Detail dropdown includes `All Operations`.
- When All Operations is selected, the detail panel displays each operation selected in the Summary, in production-flow order.
- Each operation begins with its green PASS/YIELD row, followed by ranked EC/FAIL_CODE rows.
- EC Limit applies separately to each operation.
- Selecting one operation still displays only that operation.
- Existing CMS_CONFIG first-10-character grouping, HSA/MEDIA decoding, hierarchy sorting, cumulative yield, and separate exports remain intact.
