# Yield Web App V4.9.1 - All EC Detail

Fix based on V4.9.0:
- Operation Detail displays **all EC codes by default** for each selected operation.
- `All Operations` displays all EC codes for every operation selected in the Summary.
- Optional `Top N` mode is available when faster rendering or a shorter detail table is needed.
- EC rows remain raw and do not include Recovery adjustment.
- Raw Summary, Recovery Rule Note, Recovery Summary, CMS_CONFIG first 10 characters, HSA/MEDIA decode, hierarchy sorting, caching, split rendering, debounce, and Calculating status remain unchanged.
